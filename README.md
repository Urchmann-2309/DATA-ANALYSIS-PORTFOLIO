# FOOTBALL TRANSFER MARKET ANALYSIS
## PROJECT OVERVIEW
This project analyzes global football transfer spending trends , club investment behaviour and player valuation efficiency using excel

## OBJECTIVES
- Analyze total transfer spending across 10 seasons
- Identify clubs with the highest transfer expenditure
- Detect patterns of overpayment in player transfers
- Visualize transfer trends using dashboards

## TOOLS USED
- Microsoft Excel
- Pivot Tables
- Dashboard design
  
## KEY INSIGHTS
1.  Football clubs are consistently increasing investment in players, making the transfer market a multi-billion dollar industry.
2.  There's high overvaluation of players due to competition, urgency , or hype which inflates the market.
3.  There is market dominance by european clubs which shows financial concentration amongst top teams.
4.  Transfer activity remains consistently distributed across seasons shoowing that clubs continually refresh squads rather than relying on previous squads.
